Guía para ingresar los parámetros del método

Para utilizar este método, debe proporcionar los siguientes parámetros:

  • Matriz de coeficientes (A):
    • Ingrese los valores de la matriz separados por espacios (columnas) y por punto y coma (filas).
    • Ejemplo: 2 3; 5 7 corresponde a la matriz: 
          [2, 3]
    [5, 7]
  • Vector de términos independientes (b):
    • Ingrese los valores del vector separados por espacios.
    • Ejemplo: 1 4 corresponde al vector: 
      [1, 4]
  • Vector inicial (x₀):
    • Ingrese los valores del vector separados por espacios.
    • Ejemplo: 0 0 corresponde al vector: 
      [0, 0]
  • Tolerancia:
    • Debe ser un número positivo que represente el error máximo permitido.
    • Ejemplo: 0.001.
  • Número máximo de iteraciones:
    • Debe ser un número entero positivo.
    • Ejemplo: 100.
  • Precisión:
    • Seleccione entre decimales correctos o cifras significativas.

Para más información, consulte la documentación del proyecto o contacte al administrador del sistema.
