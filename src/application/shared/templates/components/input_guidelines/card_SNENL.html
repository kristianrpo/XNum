Guía para ingresar los parámetros al método
Operadores matemáticos básicos
  • Suma (+): Usa el signo de suma entre los términos.
    • Ejemplo: x + 2
  • Resta (-): Usa el signo de resta.
    • Ejemplo: x - 5
  • Multiplicación (*): Usa el asterisco para multiplicar.
    • Ejemplo: 2 * x, x * 3
  • División (/): Usa la barra inclinada.
    • Ejemplo: x / 4, 5 / (x + 1)
  • Exponentes (**): Usa doble asterisco para elevar a una potencia.
    • Ejemplo: x ** 2 (x al cuadrado), x ** 3 (x al cubo)
Funciones Matemáticas Comunes

Debes usar las funciones matemáticas estándar de Python, como las siguientes:

  • math.sin(x): Función seno.
  • math.cos(x): Función coseno.
  • math.tan(x): Función tangente.
  • math.log(x): Logaritmo natural (base e).
  • math.log10(x): Logaritmo en base 10.
  • math.sqrt(x): Raíz cuadrada.(si se quiere hacer otra típo de raíz, utilizar operador ** para hacer la representación adecuada en vez de utilizar math.sqrt)
  • math.exp(x): Exponencial de base e (elevado a la potencia de x o de lo que se requiera).

Ejemplos:

  • math.sin(x) para el seno de x.
  • math.log(x + 1) para el logaritmo natural de (x + 1).
  • math.sqrt(x ** 2 + 1) para la raíz cuadrada de (x² + 1).
  • x ** (1/n) para calcular la raíz n-ésima de x.

Si desea conocer mas funciones, puede ingresar directamente a la documentación de la libraria Math de Python.

Uso de Paréntesis

Los paréntesis son importantes para agrupar operaciones y asegurar que se respeten las prioridades en las operaciones.

  • Correcto: (x + 2) / (x - 1)
  • Incorrecto: x + 2 / x - 1 (esto se interpretará de manera diferente a lo esperado).
Variables

Usa siempre x como la variable principal en tus funciones.

Ejemplos:

  • x ** 2 + 2 * x + 1 (función cuadrática)
  • sin(x) + log(x) (combinación de seno y logaritmo)
